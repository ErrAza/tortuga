===== Project Tortuga =====

This project is intended for internal use within Sea Monster Entertainment.

The purpose of this piece of software is to aid in the tracking and development of animation across multiple animators and art directors. The functionality at it's inception lies in the use of an Asset list and a Shot list, all information from which is stored in a SQL database hosted remotely.

Required Tools: Qt Creator 5.x and up SQLite / Azure MS-SQL

Languages Used: C++
